## PRODUCTION NOTES

### Visual Style
- **Color palette:** Dark backgrounds, vibrant accent colors (Viberr brand)
- **Typography:** Clean sans-serif, generous spacing
- **Motion:** Smooth, purposeful. No flashy transitions.
- **Product shots:** Real Viberr UI, captured via Playwright
- **B-roll:** AI-generated via Runway (office vibes, abstract tech, hands typing)

### Audio
- **VO:** Matilda (ElevenLabs) — even pacing, confident delivery
- **Music:** Minimal electronic, builds subtly. Think Odesza-lite or Tycho-adjacent.
- **SFX:** Subtle UI sounds for interactions (optional)

### Aspect Ratios
- **Hero:** 16:9 (YouTube, website) + 9:16 (Stories/Reels) + 1:1 (Feed)
- **Short-form:** 9:16 primary, 1:1 secondary
